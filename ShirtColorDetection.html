Shirt Color Detection
Generic Page

My overall goal for this project was to learn more about openCV. I decided I would give myself the simple task of using the webcam on my computer to detect the users shirt color. I achieved this by using OpenCV's built in functions to acess my webcam. I then used a face detection function to determine an approximation of the location of my face. Once the location of the face was detected I used approximate ratios of the head to body to approximate where to detect the shirt. This method does is flawed and is not meant for 100% detection it was more so an exercise to learn more about openCV
